# Portfolio – Cloud / DevOps / Cybersécurité

Portfolio technique regroupant des projets réalisés dans les domaines du Cloud Computing, de l’Infrastructure as Code, du DevOps et de la Cybersécurité.

Il met en avant une approche orientée automatisation, industrialisation des déploiements et sécurisation des infrastructures.

---

## Résumé

Ce portfolio présente une sélection de projets couvrant :

- Conception et déploiement d’infrastructures Cloud (AWS, Azure)
- Automatisation via Infrastructure as Code (Terraform)
- Mise en place de pipelines CI/CD (Jenkins, Ansible)
- Conteneurisation et orchestration (Docker, Kubernetes)
- Cybersécurité offensive et défensive (pentesting, hardening, labs)

---

## Objectifs du projet

Ce portfolio a été conçu pour :

- Centraliser les projets techniques de manière structurée
- Illustrer des compétences opérationnelles en environnement Cloud et DevOps
- Démontrer une capacité à automatiser et sécuriser des infrastructures
- Servir de support pour des candidatures techniques (DevOps / Cloud Engineer / Cybersecurity Engineer)
